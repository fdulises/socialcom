Instrucciones para instalacion manual

1-Crear una base de datos para el sitio
2-Ejecutar las consultas del archivo consultas.sql
3-Abrir el archivo config/conf_db.php y cambiar los datos por los de la base de datos
4-Subir todos los archivos a la carpeta publica del servidor
5-Ingresar al panel de administracion en la url http://TUSITIO.ALGO/admin
	Usar como usuario 'admin' y como clave 'admin'
6-En el panel dirigirse a la seccion 'sitio' y ahi cambiar la URL para que el sitio pueda funcionar
